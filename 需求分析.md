# 需求分析

\#  高校学生勤工俭学兼职服务平台

目标：生成可部署运用的服务

#

1. **服务内的角色组成**

一共涉及三个角色：

* 大学生(求职者)
* 企业方(招聘方)
* 平台运营方(管理者)

​

**1.1 大学生(求职者)**

**基础功能权限：**

* **浏览与搜索权限**：查看首页推荐、分类岗位、热门兼职列表；按行业、薪资、距离、时间筛选岗位；查看岗位详情（薪资、工作时间、地点、企业信息）
* **简历管理权限**：创建、编辑、维护个人简历（含基本信息、技能、可工作时间、课表同步）；设置简历公开/隐藏状态
* **投递与申请权限**：投递岗位申请；查看申请状态（已投递、待面试、已录用、被拒绝）；取消未处理的申请
* **沟通与面试权限**：使用平台内置IM与企业在线沟通（消息记录不可删除且在平台内留存）；发起/接受视频面试；接收面试提醒与通知
* **电子签约权限**：在线签署电子劳务协议（需阅读并同意协议条款）；查看已签署的协议列表与详情
* **任务执行权限**：通过平台打卡（签到、签退）记录工时；确认工作任务完成；对工作内容与企业进行评价打分
* **薪资与保障权限**：查看薪资明细（时薪、工时时长、应发金额、实发金额）；确认薪资到账；查看薪资担保交易记录；申请/查看平台提供的兼职意外险
* **投诉与维权权限**：发起投诉/举报（虚假岗位、薪资纠纷、人身安全问题）；上传维权证据（聊天记录、照片、协议截图）；查看投诉处理进度与结果
* **个人信息管理权限**：修改个人资料（手机号、头像、昵称）；设置隐私权限（是否公开联系方式、学历信息等）；绑定/解绑银行卡或支付账户
* **增值服务权限**（部分付费）：申请生成“兼职实践认证报告”；申请查看历史实践记录汇总

**​**

**受限/禁止权限**：

* 不得删除或伪造IM沟通记录（仅平台可审计调取）
* 不得单方面修改已签署的电子合同
* 不得发布招聘岗位信息
* 不得查看其他学生的联系方式或隐私信息
* 不得提现他人账户薪资

​

**1.2 企业方(招聘方)**

**身份定位**：长沙本地有兼职需求的企业（新消费品牌、零售连锁、上门经济等），是平台的核心付费方。

**企业认证与资质权限**：

* **入驻与认证**：提交企业注册信息（营业执照、法人身份证、联系人信息）；发起企业实名认证申请；上传资质文件（营业执照扫描件、行业经营许可证等）；查看认证审核状态与审核意见
* **信用管理权限**：查看本企业在平台的企业信用评分及变动记录；根据信用等级获得对应的权益（如更高岗位曝光、更快审核通道）

**岗位管理权限**：

* **岗位发布与管理**：创建、编辑、下架、重新发布兼职岗位（行业分类、薪资标准、工作时间段、地点、技能要求、招聘人数等）
* **岗位数据查看**：查看岗位浏览量、投递量、匹配候选人数量；查看岗位的效果分析报告（如申请者画像分析、热度分析）
* **岗位推荐与置顶**（付费功能）：购买岗位置顶、热门推荐、信息流推广等增值服务

**人才筛选与管理权限**：

* **简历浏览与筛选**：查看投递本企业的学生简历（联系方式需脱敏处理，通过平台IM沟通）；对候选人进行“感兴趣”“不合适”“待面试”等状态标记；批量下载经学生同意的简历（PDF格式）
* **人才库管理**：将优秀兼职学生加入企业人才库（需学生同意）；人才库内可按标签（技能、时间、评价）搜索；向人才库内学生发送批量兼职邀请
* **与候选人沟通**：使用平台IM与学生在线沟通；发起或接受视频面试邀请；发送/接收面试通知

**任务与结算管理权限**：

* **任务发布与确认**：确认兼职学生的到岗打卡/离岗打卡记录；对学生的工作完成情况进行确认（通过/不通过/部分完成）；对学生进行工作评价
* **薪资管理**：查看待支付薪资列表与明细（需确认学生工时与应发薪资）；发起薪资预付（资金托管至平台）；确认薪资发放成功状态；查看历史薪资支付记录与对账单
* **协议管理**：查看已签署的电子协议列表；下载电子协议副本用于企业留档

**SaaS工具与管理权限**（付费增值服务）：

* **内部管理模块**：使用在线排班工具自动生成排班表；使用自动考勤统计功能；使用薪资格式化自动核算工具；使用兼职人员档案管理（含测评、评价、考勤汇总）

**投诉与客服权限**：

* 发起投诉（学生未履约、虚假打卡等）；上传证据；查看投诉处理进度；联系平台客服寻求纠纷调解

**受限/禁止权限**：

* 不得在未经学生同意下查看其联系方式外的身份信息（如学校、身份证号部分脱敏）
* 不得随意更改已确认的工时与薪资（需双方确认或申请平台介入）
* 不得发布超出营业执照经营范围或违法的兼职岗位
* 不得将平台内学生隐私信息导出用于其他商业用途

​

**1.3 平台运营方(管理者)**

**身份定位**：平台整体运营、审核、合规监管与技术支持中心。

### 子角色1：审核管理员

**权限**：

* **企业资质审核**：查看企业提交的营业执照、法人信息；对接天眼查等第三方信用接口进行核验；审核通过/驳回企业认证申请
* **岗位信息审核**：查看待审核岗位详情（描述、薪资、地点）；对违规岗位（虚假信息、押金收费字样、违法内容）执行下架、封禁操作；标记高疑似诈骗岗位并加入黑名单
* **学生身份核验**：对身份存疑的学生申请人工复核；对接学信网接口核验学籍信息

### 子角色2：风控与安全管理员

**权限**：

* **风险监控**：查看平台实时数据看板（异常交易、高投诉岗位、批量注册账号等）；查看风控规则配置（敏感词库、触发阈值）；处理风控系统触发的预警工单
* **投诉纠纷处理**：查看所有用户（学生和企业）发起的投诉详情与证据链；进行纠纷调解、判定责任；执行资金冻结/划扣/补偿等操作；归档已结案纠纷
* **数据安全审计**：查看用户数据操作日志（谁在何时查询或导出了哪些敏感数据）；对可疑的数据导出发出告警；执行内部数据安全合规审计

### 子角色3：运营与增长管理员

**权限**：

* **用户运营管理**：查看用户统计数据（新增注册、活跃用户、留存率、转化率）；管理系统通知与推送（活动公告、岗位推荐、优惠券发放）；创建和管理校园大使团队；执行邀请奖励（积分、优惠券）发放
* **企业客户管理**：查看已签约企业客户列表与合同信息；管理企业套餐与收费配置（佣金率、广告报价、SaaS订阅报价）；为企业客户分配客户经理
* **内容管理**：管理平台安全警示教育专区内容（创建、编辑、下架）；管理社区内容（审核、置顶、删除违规帖子）

### 子角色4：财务管理员

**权限**：

* **资金账户管理**：查看与对账平台托管账户（各企业预存资金、冻结资金、可分发资金）；查看资金流水明细（每笔入账、出账、退款）；发起与银行/支付机构的资金清分指令
* **薪酬结算管理**：核对并确认每日/每周需发放的兼职薪资总表；执行批量薪资代发指令；查看分发状态（成功/失败/处理中）；处理发放失败的退款或重发
* **发票与税务管理**：查看并管理平台收取的佣金发票（开具给企业）；查看并处理由平台代扣代缴的兼职个税数据；配合税务机构提供合规数据
* **财务报表管理**：生成日/周/月度财务报告（收入、支出、毛利、回款率）；查看企业客户欠费与回款状态

### 子角色5：超级管理员/系统管理员

**权限**：

* **系统配置管理**：配置系统基本参数（时薪范围、推荐算法权重、信用评分规则等）；配置第三方API（地图、支付、短信）；管理平台字典与标签库。
* **用户角色管理**：创建/编辑/停用各子角色账户；为子角色分配权限范围；查看全站操作日志。
* **系统运维监控**：查看系统运行状态与性能指标（服务器负载、接口响应时长、错误率）；执行系统热更新与紧急回滚；管理备份策略与数据恢复。
* **合规数据查询**：调取任何必要信息用于监管部门检查或法律诉讼配合。

**受限/禁止权限**：

* 后台所有管理员均不得自行修改个人薪资与结算数据
* 运营人员不得后台直接操作资金流转（仅财务可操作）
* 审核人员不得向外部泄露审核信息或在审核期外联系企业/学生
* 所有后台操作均需留痕，接受定期审计

​

## 权限关系总结

| **角色**  | **核心活动**   | **数据私密性**        | **资金控制程度**    | **平台风险责任**       |
| ------- | ---------- | ---------------- | ------------- | ---------------- |
| 学生（C端）  | 申请、执行、评价   | 仅自己可见个人详情        | 只能查看自己的收入与流水  | 无                |
| 企业（B端）  | 发布、筛选、结算   | 仅自己可见企业信息与岗位     | 可预付并确认发放薪资    | 需要为合作学生安全负责      |
| 平台（运营方） | 审核、监管、结算执行 | 全域可见（需严格遵守权限与合规） | 可进行托管资金的调拨与清分 | 承担全平台合规风险与最终调解责任 |

**2. 服务的部署平台**

**​**

**2.1 小程序端功能模块框架**

**面向人群**：学生（C端）、企业招聘人员（B端轻量操作）

### 2.1.1 首页模块

* **LBS推荐**：基于地理位置展示“附近兼职”
* **分类导航**：行业标签（茶饮/零售/上门经济/家教）
* **热门岗位**：按热度、最新、薪资排序
* **安全警示专区**：嵌入滚动条或固定入口（反诈骗提示）
* **搜索栏**：支持岗位名称、企业名称、商圈等关键词搜索

### 2.1.2 岗位列表与详情页

* **筛选功能**：时薪范围、结算方式（日结/周结）、工作时间段、距离
* **岗位详情**：薪资说明、工作内容、地点导航、企业认证标识、信用评分
* **风险提示**：该岗位是否购买平台意外险、是否支持薪资托管

### 2.1.3 求职学生端主功能

* **简历管理**：编辑简历（技能标签、可工作时间课表同步）
* **职位投递**：一键投递、查看申请进度
* **IM即时沟通**：与企业在线交流（消息留痕不可删除）
* **视频面试**：集成SDK，支持预约面试
* **电子签约**：在线签署协议，并存证
* **打卡考勤**：GPS签到/签退，记录工时
* **薪资流水**：查看待结算/已到账薪资明细
* **评价系统**：对岗位和企业进行评价打分
* **维权入口**：发起投诉、上传证据、查看处理进度

### 2.1.4 企业招聘者端（轻量版）

* **岗位管理**：发布/编辑/下架岗位
* **人才管理**：浏览投递简历、标记状态、邀请面试
* **签约与结算**：发起电子签约、确认工时、确认薪资发放
* **沟通工具**：与学生IM交流、发送面试通知

### 2.1.5 个人中心/我的模块

* **学生**：我的简历、已投递岗位、薪资记录、协议中心、保险记录
* **企业**：企业信息管理、人才库、账单与发票
* **通用**：设置、消息中心、客服联系、实名认证管理

​

## 2.2 PC端（Web）功能模块框架

PC端分为三个子端：**学生Web拓展端**、**企业Web运营端**、**平台管理后台**

### 2.2.1 学生Web拓展端（可选）

* 简历完整编辑（富文本、PDF上传）
* 实践认证报告生成与下载
* 兼职经验社区（发帖/互动/查看他人评价）

### 2.2.2 企业Web运营端（核心使用端）

* **岗位批量管理**：Excel导入岗位、批量上下架
* **SaaS工具**：排班表管理、自动考勤统计、薪资自动核算
* **人才库管理**：搜索、标签筛选（技能、评价、历史合作次数）
* **数据看板**：岗位浏览量、投递转化率、候选人画像分析
* **财务对账**：查看佣金账单、申请开票、账户充值
* **合作协议管理**：下载已签电子协议归档

### 2.2.3 平台管理后台（运营方专用）

| **子角色**      | **主要功能**                         |
| ------------ | -------------------------------- |
| **审核管理员**    | 企业资质审核（对接天眼查）、岗位内容审核、学籍核验（对接学信网） |
| **风控/安全管理员** | 风控预警看板、投诉纠纷处理、数据访问审计日志           |
| **运营管理员**    | 用户数据统计、通知推送、校园大使管理、优惠券发放         |
| **财务管理员**    | 薪资分发指令与对账、发票管理、个税代扣缴、财务报表        |
| **超级管理员**    | 系统配置、子角色权限管理、API接口管理、运维监控        |

## 设计规范速查

### 色彩系统

```
:root {
  --color-primary: #165DFF;        /* 主色 */
  --color-primary-light: #3C7EFF;
  --color-primary-dark: #0E42D2;
  --color-primary-bg: #E8F0FE;
  
  --color-accent: #FF7D00;         /* 强调色-薪资/橙色 */
  --color-accent-light: #FF9A2E;
  
  --color-success: #00B42A;        /* 成功色 */
  --color-success-bg: #E8FFEA;
  --color-danger: #F53F3F;         /* 危险色 */
  --color-danger-bg: #FFF1F0;
  --color-warning: #FF7D00;        /* 警告色 */
  
  --color-bg: #FFFFFF;
  --color-bg-secondary: #F2F3F5;
  --color-bg-tertiary: #E5E6EB;
  --color-text: #4E5969;
  --color-text-secondary: #86909C;
  --color-text-disabled: #C9CDD4;
  --color-border: #E5E6EB;
}
```

### 字体规范
- 移动端: 基础14px, 标题16px/18px, 辅助12px, 数字20px
- PC端: 基础14px, 标题16px/18px/20px, 辅助12px
- 字体族: -apple-system, "PingFang SC", "Microsoft YaHei", sans-serif

### 间距规范
- 移动端: 页面边距16px, 卡片内边距12px, 组件间距12px
- PC端: 页面边距24px, 卡片内边距16px, 组件间距16px

### 组件规范
- 按钮: 主按钮主色填充白色文字, 圆角12px(移动端)/8px(PC端), 高度44px(移动端)/36px(PC端)
- 标签(Tag): 圆角4px, 内边距4px 8px
- 卡片(Card): 白底圆角8px, 阴影0 2px 8px rgba(0,0,0,0.08)
- 输入框: 高度44px(移动端)/36px(PC端), 边框色#E5E6EB, 获焦色#165DFF

**内容+数据来源**:
- 安全警示: 系统配置 `t_system_config` 表，静态文案
- 搜索栏: 点击跳转搜索页（placeholder静态）
- 分类标签: 从字典表加载，选中状态本地管理
- 岗位列表: `GET /v1/student/job/list?longitude=&latitude=&distance=5&page=1&size=20`
- 底部安全提示: 静态文案

**交互状态**:
- 加载中: 骨架屏(skeleton)占位，3条卡片骨架
- 空数据: 空状态插图 + "暂无附近岗位" + "扩大搜索范围"按钮
- 未定位: 定位权限引导弹窗，"开启定位，查看附近兼职"按钮
- 网络错误: "网络异常，点击重试"居中显示
- 下拉刷新: 重新请求page=1，下拉动画
- 上滑加载更多: 滚动到底部自动加载下一页

**间距和颜色**:
- 页面左右边距: 16px
- 卡片间距: 12px
- 卡片内边距: 12px
- 安全警示条: 背景#FFF1F0，文字#F53F3F，高度40px
- 搜索栏: 背景#F2F3F5，高度44px，圆角22px
- 分类标签: 选中蓝底#165DFF白字，未选灰底#F2F3F5灰字

- 卡片圆角: 8px
- 薪资颜色: #FF7D00
- 安全保障卡: 边框#00B42A，背景#E8FFEA

## 五、通用状态说明

### 加载状态
- **骨架屏**: 页面主要区域显示灰色占位块，模拟内容布局
- **Loading遮罩**: 提交操作时全屏半透明遮罩 + 加载动画
- **局部加载**: 按钮内显示小菊花 + 文字变"加载中..."

### 空数据状态
- 空状态插图 (平台统一插画)
- 提示文字（如"暂无数据"、"还没有投递记录"）
- 操作按钮（如"去逛逛"、"去发布"）

### 错误状态
- 网络错误: 插图 + "网络异常，点击重试"
- 接口错误: Toast 提示错误信息
- 权限不足: 403页面 + "无权限访问" + "返回首页"

### 确认弹窗
- 标题 + 内容 + [取消] [确定] 按钮
- 危险操作: 确定按钮红色
- 普通操作: 确定按钮主色

### Toast提示
- 成功: 绿色对勾 + 文字
- 错误: 红色感叹号 + 文字
- 警告: 黄色感叹号 + 文字
- 时长: 2秒自动消失

## A. 投递成功动效

### 基本信息

| 项目 | 参数 |
|------|------|
| **触发条件** | 点击"立即投递"按钮且接口返回成功 |
| **总时长** | 1500ms |
| **适用页面** | 岗位详情页 `pages/student/job-detail` |
| **实现方式** | CSS @keyframes + transform + JS控制时序 |

### 动画序列（时间轴）

```
0ms      ──── 按钮按下缩小
100ms    ──── 按钮变绿色
300ms    ──── 简历图标飞出，沿弧线上升
900ms    ──── 企业图标接收弹跳
1200ms   ──── 绿色对勾放大出现
```

```css
  0% { transform: scale(1); }

}
```

- 缓动: ease-in

#### Step 2: 按钮变绿色

```css

  0% { background-color: #165DFF; }
  100% { background-color: #00B42A; }
}
```

- 时长: 200ms
- 缓动: ease-out
- 颜色值: 成功绿 #00B42A

#### Step 3: 简历图标弧线飞出

| **适用页面** | 薪资流水页 `pages/student/salary/index` |
| **实现方式** | CSS animation + requestAnimationFrame 数字滚动 |

### 动画序列

```

500ms    ──── 金币落地，开始弹跳
900ms    ──── 弹跳结束，金额开始滚动
1700ms   ──── 数字滚动完成，开始闪烁
2000ms   ──── 闪烁结束，动画完成
```

### 详细参数

#### Step 1: 金币下落 + 旋转

```css
@keyframes coinFall {
  0% {
    transform: translateY(-200px) rotate(0deg);
    opacity: 0;
  }
  10% {
    opacity: 1;
  }
  100% {
    transform: translateY(0) rotate(360deg);
    opacity: 1;
  }
}
```

- 时长: 500ms
- 缓动: cubic-bezier(0.55, 0.055, 0.675, 0.19) (ease-in)
- 金币大小: 64px
- 起始位置: 屏幕顶部外 -200px
- 金币颜色: 金色渐变 #FFD700 → #FFA500

#### Step 2: 金币弹跳

```css
@keyframes coinBounce {
  0%, 100% { transform: translateY(0); }
  25% { transform: translateY(-30px); }
  50% { transform: translateY(0); }
  75% { transform: translateY(-15px); }
}
```

- 时长: 400ms
- 缓动: cubic-bezier(0.28, 0.84, 0.42, 1)
- 弹跳2次: 第一次高30px，第二次高15px

#### Step 3: 金额数字滚动（countUp 算法）

```javascript
/**
 * 数字滚动动画
 * @param {HTMLElement} element - 目标元素
 * @param {number} target - 目标金额
 * @param {number} duration - 动画时长(ms)
 */
function countUp(element, target, duration = 800) {
  const startTime = performance.now();
  const startValue = 0;
  const decimals = 2;

  function update(currentTime) {
    const elapsed = currentTime - startTime;
    const progress = Math.min(elapsed / duration, 1);

    const easeProgress = 1 - Math.pow(1 - progress, 3);
    const currentValue = startValue + (target - startValue) * easeProgress;

    element.textContent = currentValue.toFixed(decimals);

    if (progress < 1) {
      requestAnimationFrame(update);
    }
  }

  requestAnimationFrame(update);
}
```

- 时长: 800ms
- 缓动: easeOutCubic
- 字体大小: 32px
- 字重: bold
- 颜色: #FF7D00
- 前缀: ¥ 符号，同步显示

#### Step 4: 金额闪烁

```css
@keyframes amountGlow {
  0%, 100% {
    text-shadow: 0 0 0 rgba(255, 125, 0, 0);
    color: #FF7D00;
  }
  50% {
    text-shadow: 0 0 20px rgba(255, 125, 0, 0.8);
    color: #FF9500;
  }
}
```

- 时长: 300ms × 3次 = 900ms
- 循环: 3次
- 闪烁颜色: 从 #FF7D00 到 #FF9500

## E. 页面过渡动效

### 基本信息

| 项目 | 参数 |
|------|------|
| **适用范围** | 所有页面切换、弹窗显示/隐藏 |
| **实现方式** | CSS transition + Vue/uni-app 过渡组件 |

### E.1 页面进入/退出

#### 页面进入（从右向左滑入）

```css
.page-enter {
  transform: translateX(100%);
  opacity: 0;
}
.page-enter-active {
  transform: translateX(0);
  opacity: 1;
  transition: all 300ms cubic-bezier(0.16, 1, 0.3, 1);
}
```

- 时长: 300ms
- 缓动: ease-out-expo
- 位移: 从右侧100%到0
- 透明度: 0 → 1

#### 页面退出（从左向右滑出）

```css
.page-leave {
  transform: translateX(0);
  opacity: 1;
}
.page-leave-active {
  transform: translateX(-30%);
  opacity: 0;
  transition: all 200ms cubic-bezier(0.32, 0, 0.67, 0);
}
```

- 时长: 200ms
- 缓动: ease-in
- 位移: 从0到左侧30%（轻微移出，制造景深）
- 透明度: 1 → 0

### E.2 弹窗/面板

#### 弹窗出现（底部弹出）

```css
.modal-enter {
  transform: translateY(100%);
  opacity: 0;
}
.modal-enter-active {
  transform: translateY(0);
  opacity: 1;
  transition: all 300ms cubic-bezier(0.16, 1, 0.3, 1);
}
```

- 时长: 300ms
- 缓动: ease-out-expo
- 适用: 底部弹出面板、筛选面板、ActionSheet

#### 弹窗出现（中心缩放）

```css
.modal-center-enter {
  transform: scale(0.8);
  opacity: 0;
}
.modal-center-enter-active {
  transform: scale(1);
  opacity: 1;
  transition: all 200ms cubic-bezier(0.16, 1, 0.3, 1);
}
```

- 时长: 200ms
- 缓动: ease-out
- 初始缩放: 0.8
- 适用: 居中对话框、确认弹窗

#### 弹窗消失

```css
.modal-leave-active {
  transform: scale(0.9);
  opacity: 0;
  transition: all 150ms cubic-bezier(0.32, 0, 0.67, 0);
}
```

- 时长: 150ms
- 缓动: ease-in
- 缩放到: 0.9
- 透明度: 0

### E.3 遮罩层过渡

## M. 全局缓动函数规范

### 标准缓动曲线

| 名称 | 贝塞尔值 | 适用场景 |
|------|---------|---------|
| **ease-out-expo** | `cubic-bezier(0.16, 1, 0.3, 1)` | 出场动画、元素进入、页面滑入 |
| **ease-in-expo** | `cubic-bezier(0.32, 0, 0.67, 0)` | 退场动画、元素消失、页面滑出 |
| **ease-in-out** | `cubic-bezier(0.645, 0.045, 0.355, 1)` | 对称动画、循环动画 |
| **bounce** | `cubic-bezier(0.68, -0.55, 0.27, 1.55)` | 弹性反馈、成功提示、收藏点赞 |

### 动画时长规范

| 类型 | 时长 | 示例 |
|------|------|------|
| **微交互** | 100ms | 按钮按下、图标点击 |
| **快速过渡** | 150-200ms | 弹窗消失、Tab切换 |
| **标准过渡** | 250-300ms | 页面切换、弹窗出现 |
| **强调动画** | 400-600ms | 成功动效、金币下落 |
| **长动画** | 800-1500ms | 数字滚动、复杂序列 |
| **循环动画** | 1.5s-3s | 骨架屏、加载中、空状态浮动 |

## 实现建议

### 1. CSS 变量统一管理

```css
:root {
  --color-primary: #165DFF;
  --color-success: #00B42A;
  --color-warning: #FF7D00;
  --color-danger: #F53F3F;

  --ease-out-expo: cubic-bezier(0.16, 1, 0.3, 1);
  --ease-in-expo: cubic-bezier(0.32, 0, 0.67, 0);
  --ease-bounce: cubic-bezier(0.68, -0.55, 0.27, 1.55);

  --duration-fast: 150ms;
  --duration-normal: 300ms;
  --duration-slow: 500ms;
}
```

### 2. 性能优化

- 优先使用 `transform` 和 `opacity` 做动画（触发合成层，不重排）
- 避免使用 `width`/`height`/`top`/`left` 做动画
- 复杂动画使用 `will-change: transform` 提前声明
- 移动端注意功耗，避免过多同时运行的动画

### 3. 无障碍支持

```css
@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}
```
